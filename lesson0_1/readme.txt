# 从传感器的选型，realsense不仅能够提供可见光图像；当然还有红外，以及三维点云；
# 我在传感器使用的后半程，加入了open3d初步可视化和数据采集，其目的与之前类似。
# 在视听触味嗅当中，听觉以及对声音的记录在19世纪中叶开始起步，同一时期，银盐相机诞生，使得我们能够记录黑白照片，半个世纪后彩色相机诞生。
# 直到今天，相机已经成为手机的必备功能，和对标友商的对比参数之一。
# 另一个有意思的事情在于，apple实际上在一系列红外深度相机、结构光、memos激光之后，在2021年左右在ipad和高端手机也集成了深度传感器。
# 如果说相机使得我们能够采样和处理三维世界的二维投影，那么三维数据的采集和计算，可以预见是一个新的环境感知时代的来临。
# 而超出三维的环境感知，除了时空融合之外，高维高斯空间和非高斯空间均存在于数学模型中。
# 考虑传感器的价格、体积、重量和可操作性，课程中还是主要以感知距离相对有限，但是体积和便携性更高的realsense为主。
# 同样，希望通过若干的小的例程，使得学生能够初步了解和掌握三维数据，包括点云，着色点云的采集和基本数据结构。
# 从而推动他们去接触这一新兴的技术领域，为后续课程的算法实测提供数据源。
# 更为重要的是，通过在初期引入传感器使用，能够在整个学期完善和优化数据采集流程，因为任何涉及硬件和物理世界交互的系统的复杂度、问题和困难都要远高于纯粹的理论建模。
